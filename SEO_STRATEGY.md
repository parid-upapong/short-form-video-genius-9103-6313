# SEO Strategy: Dominating the "AI Video" Search Landscape

## 1. Core Keyword Pillars
We will target three primary search intent clusters:
- **Product Specific:** "AI video generator", "Text to video AI", "Automated short-form video".
- **Platform Specific:** "TikTok automation tools", "How to make viral Reels", "AI for YouTube Shorts".
- **Niche/Pain Point:** "Faceless YouTube channel ideas", "Viral video script generator", "Video editing for non-editors".

## 2. Programmatic SEO (P-SEO) Plan
We will generate 500+ landing pages targeting "niche + AI video".
- **URL Structure:** `overlord.ai/tools/ai-video-for-[niche]`
- **Examples:** 
    - `/tools/ai-video-for-real-estate`
    - `/tools/ai-video-for-crypto-news`
    - `/tools/ai-video-for-motivational-quotes`

## 3. Technical SEO Requirements
- **Server-Side Rendering (SSR):** Using Next.js for fast indexing.
- **JSON-LD Schema:** Implement `SoftwareApplication` and `HowTo` schema for video creation steps.
- **Core Web Vitals:** Target < 1.2s LCP (Largest Contentful Paint) via static asset optimization.

## 4. Backlink Strategy (The "Creator Loop")
- **Watermark Credits:** Free tier videos include a "Created by Overlord.ai" watermark with a unique short-link.
- **Integration Directory:** Get listed on AI directories (Futurepedia, There's An AI For That).
