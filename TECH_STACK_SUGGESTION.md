# Recommended Tech Stack for Implementation

| Component | Technology |
| :--- | :--- |
| **Frontend** | Next.js 14 (App Router), Tailwind CSS, Framer Motion (for smooth transitions) |
| **Backend** | Node.js / Python (FastAPI for AI heavy lifting) |
| **Database** | PostgreSQL (Supabase) for user data/metadata |
| **LLM** | GPT-4o or Claude 3.5 Sonnet (Scripting & Logic) |
| **Video Engine** | Remotion (React-based video) or Shotstack API |
| **Voice** | ElevenLabs API / OpenAI Whisper (Transcription) |
| **Storage** | AWS S3 or Cloudflare R2 |
| **Queueing** | Redis + BullMQ (to handle rendering bottlenecks) |
